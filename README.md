Assignment 2 — Support Ticket Automation Workflow (QuickCart)
**Tools Used: Google Forms | Google Sheets | n8n | Gmail**

Built an end-to-end automation system for QuickCart to automatically route 
customer support tickets to the correct team without manual intervention.

 Automation Workflow Built:
- 📋 **Google Forms** — Customer submits support request
- 📊 **Google Sheets** — Form response stored as a new ticket
- 🔁 **n8n** — Detects new row and checks ticket category
- 🔀 **Routing Logic** — Ticket routed to correct support team
- 📧 **Gmail** — Automated acknowledgement email sent to customer
